Bowling Scoring System
==========

Requirements
----
-  [x] has 10 frames
-  [x] takes input from terminal
-  [x] spares and strikes calculate
-  [x] current score displays at end of each round
-  [x] final score of player displays at the end of the game

Technologies used

Setup
```sh
$ git clone this repo
$ cd tenpinbowling
$ npm install
```

Run
```sh
$ node index.js
```
